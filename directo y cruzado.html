CABLEADO DIRECTO

El cable directo, también conocido como cable recto, se utiliza para conectar dispositivos diferentes, como un ordenador a un switch o un router a un switch. Los cables directos utilizan el mismo esquema de pinout en ambos extremos.

CABLEADO CRUZADO

El cable cruzado se utiliza para conectar dispositivos similares, como dos ordenadores directamente o dos switches entre sí. En un cable cruzado, los pares de cables están conectados de manera cruzada en ambos extremos para permitir la comunicación directa entre dispositivos similares.
